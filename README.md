# 🔐 Modern Password Generator

A sleek, interactive web application that generates secure, randomized passwords. This project focuses on a clean user interface and customizable security parameters, built using Javascript.

 ## ✨ Features

* **Real-time Length Control:** Adjust password length (up to 20+ characters) using an intuitive range slider.
* **Granular Customization:** Toggle specific character sets:
    * Uppercase Letters (A-Z)
    * Lowercase Letters (a-z)
    * Numbers (0-9)
    * Special Symbols (@!#$^* )
* **One-Click Copy:** Integrated copy-to-clipboard functionality with a visual icon for seamless use.
* **Dynamic UI:** Clean, centered glassmorphism-style container with a responsive layout.

## 🛠️ Built With

* **HTML5:** Structured for accessibility and clear input labeling.
* **CSS3:** Custom styling featuring soft glows, centered positioning, and a professional color palette.
* **JavaScript (ES6):** Logic for randomizing character selection based on user-defined constraints.

## 💡 Usage

1.  Use the **slider** to set your desired password length.
2.  Check the boxes for the **character types** you wish to include.
3.  Click the **Generate Password** button.
4.  Click the **Copy Icon** next to the output box to save it to your clipboard.
